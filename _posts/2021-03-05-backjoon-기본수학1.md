### 10250 ACM 호텔

    h, w, n = map(int, input().split())
    f = 0
    ho = 0

        f = h * 100

### 2775 ~~부녀회장이 될테야~~ ( 시간초과 )
```python
num = int(input())

def fun(a, b):
    if a == 0:
        return b
        
    else:
        s = 0
        for i in range(1, b+1):
            s += fun(a-1, i)
        return s

for i in range(num):
    k = int(input())
    n = int(input())
    print(fun(k, n))
```

### 1011 Fly me to the Alpha Centauri
```python
t = int(input())
for i in range(t):
    a, b = map(int, input().split())
    c = b - a
    num = 1
    while True:
        if num ** 2 <= c < (num + 1) ** 2:
            break
        num += 1
    if num ** 2 == c:
        print((num * 2) - 1)
    elif num ** 2 < c <= num ** 2 + num:
        print(num * 2)
    else:
        print((num * 2) + 1)
```
