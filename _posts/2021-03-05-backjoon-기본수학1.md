### ACM 호텔
```python
t = int(input())
for i in range(t):
    h, w, n = map(int, input().split())
    f = 0
    ho = 0
    if n % h == 0:
        f = h * 100
        ho = n // h
    else:
        f = (n % h) * 100
        ho = 1 + n // h
    print(f + ho)
```
### ~~부녀회장이 될테야~~

```python
num = int(input())

def fun(a, b):
    if a == 0:
        return b
        
    else:
        s = 0
        for i in range(1, b+1):
            s += fun(a-1, i)
        return s

for i in range(num):
    k = int(input())
    n = int(input())
    print(fun(k, n))
```
