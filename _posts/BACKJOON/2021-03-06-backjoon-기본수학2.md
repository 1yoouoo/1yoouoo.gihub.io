### 1978 소수 찾기
```python
n = int(input())
sosu = list(map(int, input().split()))
def prime(num):
    if num == 1:
        return False
    elif num == 2:
        return True
    for i in range(2, num):
        if num % i == 0:
            return False
    return True
count = 0
for i in sosu:
    if prime(i):
        count += 1
print(count)
```
