### 정수 N개의 합
```python
def solve(num_list):
    result = 0
    for num in num_list:
        result += num
    return result
```
### 셀프 넘버
```python
def d(n):
    a = 0
    for x in list(str(n)):
        a = a + int(x) 
    return int(n) + a
a= []

    a.append(k)
```

### 한수
```python
n = int(input())
han = 0
for i in range(1, n + 1):
    if i < 100:
        han += 1
    else:
        ns = list(map(int, str(i)))
        if ns[0] - ns[1] == ns[1] - ns[2]:
            han += 1
print(han)
```
